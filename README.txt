== Description ==

Write notes for orders with this plugin. Sometimes you need to write a note or explanation for orders, you can do it easily using WooCommerce EasyNote.
IT IS FREE

Features

*   Display the number of order notes in the order list table
*   The possibility of recording notes in the preview pop-up
*   It is possible to remove the note from the order preview screen
*   Show the last 5 notes in the order preview popup

== Installation ==

1-	Our plugin (EasyNote) won’t be working without WooCommerce plugin. Make sure WooCommerce plugin is installed and activated.
2-	Upload the plugin files to the /wp-content/plugins directory, or install the plugin through the WordPress ‘Plugins’ screen directly.
3-	Activate the plugin through the Plugins menu in WordPress
